Результат подписи

Как использовать офлайн

  1. Установите плагин CryptoPro Browser Plug-in и убедитесь, что в системе есть cadesplugin_api.js.
  2. Скопируйте файл cadesplugin_api.js в корень рядом с index.html (этот файл уже добавлен).
  3. Запустите приложение через локальный сервер: npm start и откройте http://localhost:8080.
  4. Нажмите «Обновить», выберите сертификат и подпишите нужный файл.
Поддерживаются плагины CryptoPro с объектной моделью CAdESCOM/CAPICOM.
