Подписание файлов с помощью плагина CryptoPro

Легковесное одностраничное приложение для работы полностью локально в браузере. Требуется установленный плагин CryptoPro (или совместимый) и файл vendor/cadesplugin_api.js, поставляемый вместе с плагином.

Файл и параметры

Используется формат CAdES-BES. Отсоединённая подпись не содержит исходный файл.

Результат подписи

Как использовать офлайн

  1. Установите плагин CryptoPro Browser Plug-in и убедитесь, что в системе есть cadesplugin_api.js.
  2. Скопируйте файл cadesplugin_api.js в каталог vendor/ рядом с этим приложением.
  3. Откройте файл index.html локально в браузере (Chrome/Edge/Firefox) с разрешённым плагином.
  4. Нажмите «Обновить», выберите сертификат и подпишите нужный файл.
Поддерживаются плагины CryptoPro с объектной моделью CAdESCOM/CAPICOM.
