# Офлайн-приложение для подписи файлов ЭЦП

Легковесное веб-приложение, работающее локально в браузере и использующее установленный плагин CryptoPro (или совместимый) для подписи файлов.

## Быстрый старт

> Важно: запуск по `file://` отключает интеграцию с CryptoPro. Используйте локальный HTTP-сервер.

1. Установите CryptoPro Browser Plug-in и убедитесь, что у вас есть файл `cadesplugin_api.js`.
2. Скопируйте `cadesplugin_api.js` в каталог `vendor/` рядом с `index.html`.
3. Установите зависимости (для запуска сервера используются только встроенные модули Node.js):
   ```bash
   npm install
   npm start
   ```
4. Откройте приложение в браузере по адресу [http://localhost:8080](http://localhost:8080).
5. Нажмите «Обновить», выберите сертификат с закрытым ключом и подпишите файл.

## Особенности
- Формат подписи: CAdES-BES.
- Поддержка отсоединённой подписи (опция «Создать отсоединённую подпись»).
- Выгрузка результата в Base64 и скачивание `.sig` файла.
- Полностью офлайн: никаких сетевых запросов со стороны приложения (кроме загрузки плагина из локального каталога `vendor/`).
