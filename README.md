# Офлайн-приложение для подписи файлов ЭЦП

Легковесное веб-приложение, работающее локально в браузере и использующее установленный плагин CryptoPro (или совместимый) для подписи файлов.

## Быстрый старт

1. Установите CryptoPro Browser Plug-in и убедитесь, что у вас есть файл `cadesplugin_api.js`.
2. Скопируйте `cadesplugin_api.js` в каталог `vendor/` рядом с `index.html`.
3. Откройте `index.html` в поддерживаемом браузере (Chrome/Edge/Firefox) с разрешённым плагином.
4. Нажмите «Обновить», выберите сертификат с закрытым ключом и подпишите файл.

## Особенности
- Формат подписи: CAdES-BES.
- Поддержка отсоединённой подписи (опция «Создать отсоединённую подпись»).
- Выгрузка результата в Base64 и скачивание `.sig` файла.
- Полностью офлайн: никаких сетевых запросов со стороны приложения.
